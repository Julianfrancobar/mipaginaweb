HARDWARE

corresponde a todas las partes tangibles de un sistema
informático. Sus componentes son: eléctricos, electró-
nicos, electromecánicos y mecánicos. Utilizado para la
salida de información, el monitor es el canal por donde
la computadora muestra la información al usuario.

COMPONENTES DEL HARDWARE

  • unidad central de procesamiento(cpu)
  • placa base
  • memoria ram
  • memoria rom
  • memoria cache
  • fuente de alimentacion
  • disco duro
  • monitor,raton y teclado
